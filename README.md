# NYC_Yellow_Taxi_Trip_Analysis_2024

Analysis of the NYC Yellow Taxi 2024 dataset, exploring trip patterns, fares, tipping behavior, airport trends, and borough-level insights using Python and data visualization techniques.

#Project Overview

The goal of this project is to clean, process, and analyze the 2024 Yellow Taxi trip data, then present key findings through visualizations and a technical report. The project highlights how New York City moves and how rider behavior changes across time, geography, and trip types.

#Dataset

Source: NYC Taxi & Limousine Commission (TLC)
Format: Monthly parquet files for 2024
Contains fields such as timestamps, distance, fare amounts, surcharges, payment types, and zone IDs.
